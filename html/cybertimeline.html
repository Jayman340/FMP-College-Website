Cyber Threat Timeline

ILOVEYOU (2000)

Type: Email worm

Vector: Email attachment

This virus spread through emails pretending to be from someone you know.

Learn More

Mydoom (2004)

Type: Worm

Vector: Email and network spread

One of the fastest spreading worms at the time. It was used to overload websites and cause disruption.

Learn More

Conficker (2008)

Type: Worm / Botnet

Vector: Weak passwords and vulnerabilities

Conficker created a huge botnet by exploiting weak security on computers and networks.

Learn More

Stuxnet (2010)

Type: Cyber-weapon

Vector: Infected USB / Zero-day Exploits

Stuxnet was specifically engineered to cause damage to Iranian uranium enrichment facilities.

Learn More

WannaCry (2017)

Type: Ransomware

Vector: Unpatched systems

This attack locked files and demanded payment. It affected hospitals and organisations worldwide.

Learn More

NotPetya (2017)

Type: Wiper malware

Vector: Software update exploit

It looked like ransomware but actually destroyed data. It spread rapidly through company networks.

Learn More

SolarWinds (2020)

Type: Supply chain attack

Vector: Third-party software

Hackers accessed organisations through a trusted software update, affecting governments and businesses.

Learn More

Type: Data extortion

Vector: Data theft and threats

Attackers steal data and threaten to release it unless payment is made, increasing pressure on victims.
